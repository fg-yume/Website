Death Race Survivor

Death Race Survivor is a level design creation made in Minecraft using WorldEdit. The goal behind Death Race Survivor was to develop and test level design. Various thematic levels were created, each with different levels of difficulty, in order to create new and exciting changes for the player as they made progression through the game.

Specifications
Game Minecraft 1.7.4
Technology WorldEdit
Team Size 3

Despair

Despair is a design for a strategy role-playing game that was heavily tested using a paper prototype. Statistics and abilities were created for the various playable classes and enemies in the game. An iterative design approach was taken in order to create a fun, yet challenging experience for players.

Specifications
Prototype Paper
Team Size 3
