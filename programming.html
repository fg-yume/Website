Food Friends

Mobile controls are currently being worked on

Food Friends is a casual game built using Javascript and the canvas API. In order to keep true to the game jam's theme of "transatlantic friendship", Food Friends' core mechanic is to control various different countries into eating food that is not associated with their own country. If a country consumes food that is associated with them, they grow fatter. After consuming a certain amount of food from their own country, a country becomes unable to eat and is removed from play. The game will end if all available characters are removed from play.

Specifications
Language Javascript
API HTML5 Canvas
Team Size 8

Chocofling

Chocofling is a casual game built using the Sprite Kit framework. The goal behind Chocofling is to accrue as many points as possible by coating falling food with fondue. The fondue supply is limited; the game is over once the supply is depleted. Non-food objects will also fall, and will negatively impact the players' score if coated with fondue. In addition, powerups will also spawn, changing the dynamics of the game.

Specifications
Language Objective-C
Technology Xcode 5.0
Framework Sprite Kit
Team Size 3

Moto

Moto is a naval real time strategy game built using DirectX 11 and C++. Moto was designed to be simple at its core, while at the same time being able to scale easily. This model allowed the team to learn more about graphics programming while implementing the new knowledge into the game.

Specifications
Language C++
Technology Visual Studio 2012
API DirectX 11
Libraries OpenAL Soft
Team Size 5

Break[in]

Break[in] is a small game built using C++ and OpenGL. Similar to the original breakout, the goal of Break[in] is to direct a ball to destroy all of the targets in a level. Sound effects, as well as different background music for every game state, was implemented in Break[in] in order to extend familiarity with various libraries.

Specifications
Language C++
Technology Visual Studio 2012
Libraries OpenGL 3.2 | OpenAL Soft | GLFW | GLEW | GLM
Team Size 2
